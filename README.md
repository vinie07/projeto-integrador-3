Os alunos, organizados em equipes, deverão desenvolver uma solução computacional para coleta, integração, tratamento e visualização de dados públicos sobre Síndrome Respiratória Aguda Grave (SRAG) no Brasil.

A base de dados deverá ser obtida no Portal de Dados Abertos do SUS: Banco de dados da Síndrome Respiratória Aguda Grave (SRAG) – 2019 a 2026. A equipe deverá utilizar os arquivos em formato CSV, considerando obrigatoriamente os anos de 2021 a 2026.

A primeira etapa consiste em **criar um shell script capaz de baixar, organizar e integrar os arquivos**CSV dos anos selecionados em um único arquivo de log/dataset consolidado. O script deverá tratarcabeçalhos, padronizar a estrutura dos dados e gerar um arquivo final organizado para análise.

Na segunda etapa, a equipe deverá utilizar recursos de desenvolvimento em Linguagem C e/ou Python, associados ao GNUplot ou ferramenta semelhante, para gerar gráficos e visualizações  
sobre os dados.

As análises podem considerar, por exemplo: número de casos por ano, distribuiçãopor estado, evolução temporal, faixa etária, sexo, classificação final, internações, óbitos ou outros indicadores relevantes presentes no dataset.

Como produto final, a equipe deverá apresentar um dashboard com os principais resultados da análise. Preferencialmente, esse dashboard deverá ser disponibilizado em um portal web, permitindo que o usuário visualize os gráficos e compreenda os dados de forma clara e interativa.

Entregáveis:

1.  Shell script utilizado para baixar, organizar e integrar os arquivos CSV.
    
2.  Arquivo consolidado de log/dataset com os dados de 2021 a 2026.
    
3.  Código-fonte em C e/ou Python utilizado para tratamento e geração das visualizações.
    
4.  Scripts ou comandos utilizados no GNUplot ou ferramenta equivalente.
    
5.  Dashboard funcional ou protótipo web com os gráficos gerados.
    
6.  Relatório técnico contendo descrição da solução, etapas realizadas, dificuldades encontradas e análise dos principais resultados.
    

Critérios de avaliação:  
• Correta utilização de shell script para automação da integração dos dados.  
• Organização, limpeza e padronização do arquivo consolidado.  
• Uso adequado de C, Python, GNUplot ou ferramentas similares.  
• **Qualidade dos gráficos e clareza das visualizações.  
• Funcionalidade e apresentação do dashboard.  
• Coerência da análise dos dados e qualidade do relatório técnico.  
• Organização do código, documentação e trabalho em equipe.**
